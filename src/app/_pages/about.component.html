Campaign Supernova provides a simple user interface to devise and brief layouts for email campaign templates; store and edit all your assets; and get layouts and assets reviewed and approved by your team...

Faq

So this isn't an email build tool?

CS functions as a companion tool to an email builder, specifically one that generates and populates email templates based on a set of reference data.Each time you compose a layout and collate assets for an email you are not just coordinating your resources but compiling the data which can be directly referenced to build that email.

I want an email build tool!

Campaign Builder is a ‘one click’ email builder that can do the rest.* The modules available in Campaign Supernova's UI correspond to html modules available in Campaign Builder's library. If you want to build a template based on the campaign data you have set up in CS - then Campaign Builder can make that happen.

Are we done? Any more patronising mansplanation?

Whether briefing your own or another organisation CS can help maximise workflow and minimise errors that occur when working with static briefing documents; exchanging wireframes and assets via email or server; collating feedback and changes over email chain or hosted documents. Keep scrolling to learn more about its features and controls before you dive in.

Features

Compose

Compose bespoke emails

Use the editor to devise custom layouts based on pre-built html modules

Configure

Configure styles and branding

e.g. Select from preset options for button and background colours

Compile

Submit and manage assets

Upload copy; headings; T&Cs; ctas; links; images; inline links and assign them to prebuilt html modules

Catch Errors

Filter out errors with validation

Custom validators may be applied to any asset type e.g. valid url or image size

Optimise

Text encoding all done for you

Any copy submitted is automatically encoded for email rendering

Review

Approval controls aid workflow

Campaign assets, styles and layout can be assembled, reviewed and approved all through one app and coms stream

Campaign Supernova allows registered users to be assigned one of three different types of profile when initiating a campaign. Each of these profiles have a unique set of privileges when accessing a campaign...

Controls

Author

  • • Create, edit or delete campaign modules
  • • Request QA
  • • Request final approval from the owner

QA

  • • Approve or reject campagin modules
  • • Apply a red flag to any data field FAO the author
  • • Register feedback for any module

Owner

  • • Approve or reject a campaign
  • • Register feedback regarding the campaign
  • • Apply a red flag to any data field or module FAO the author

Try it
